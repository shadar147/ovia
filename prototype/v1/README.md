# Ovia Clickable HTML Prototype (v1)

Laptop-first clickable prototype.

## Run locally
From repo root:

```bash
cd prototype/v1
python3 -m http.server 8080
```

Open:
- http://localhost:8080

- dashboard.html
- team-identity.html
- person-360.html
